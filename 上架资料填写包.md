### 描述（Description，≤4000字）
```
战区预警是一款面向普通民众的应急预警与避险辅助工具。它聚合公开来源的空袭、炮击、灾害等风险信息，按你所在位置推送预警，并提供避难所导航、离线地图与一键求助，帮助你和家人在紧急时刻更快做出避险决策。

【主要功能】
• 实时预警：根据你的位置，推送附近的空袭 / 炮击 / 灾害风险提示，争取宝贵的避险时间。
• 避难导航：地图标注附近避难所与安全点，帮助规划撤离路线。
• 离线地图：预先下载所在区域地图，断网也能查看避难所与路线。
• 一键 SOS：紧急时向你预先设置的紧急联系人发送带定位的求助（短信 / 语音 / 推送）。
• 家人互助：建立家庭群组，共享安全状态，互相确认平安。
• 模拟演习：内置演习模式，平时即可熟悉预警与避险的完整流程。

【重要说明】
• 本应用为民用安全参考工具，所聚合信息来自公开来源，并非官方权威发布渠道，也无法保证信息的实时性、完整性或绝对准确。
• 紧急情况下，请以当地官方预警与应急部门指引为准，本应用仅作辅助参考。
• 一键 SOS 的发送对象为你自行设置的紧急联系人，本应用不代为拨打当地紧急电话。

【订阅说明】
• 高级功能通过自动续期订阅提供：【你来填：例如 解锁更多预警区域 / 更精细地图图层 / 家庭成员扩容 等】。

如有任何问题或建议，欢迎通过支持页面与我们联系。
```
