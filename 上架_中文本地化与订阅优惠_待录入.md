## 二、订阅优惠（两条订阅各配 2 个）

> 入口：左侧「订阅」→ 点开订阅（Personal Monthly / Family Monthly）→ 找「介绍性优惠 / Introductory Offers」「促销优惠 / Promotional Offers」两块。

### ① 介绍性优惠（新人首月半价）——无需 .p8，建好即生效
- 类型：**按需付费 (Pay as you go)** 或 **预付费**；
- 时长：**1 个月**；
- 价格：选最接近**基础价 50%** 的价位 —— **Personal ≈ $19.99**、**Family ≈ $49.99**（或改为「7 天免费试用」）；
- 适用：新订阅用户；销售范围：全部（沿用订阅的 174 国）。

### ② 促销优惠（推荐返券）——标识符必须 `referral50`
- **标识符（Offer Code/Identifier）**：必须填 **`referral50`**（后端代码就按这个名字签名）；
- 类型：按需付费；时长：**1 个月**；价格：**基础价 50%**（同上 $19.99 / $49.99）；
- ⚠️ **运行期依赖你的 .p8 签名密钥**：要先在「用户和访问 → 集成 → App 内购买」生成密钥，记下 Key ID + 下载 .p8，设为 Supabase 密钥 `APPSTORE_OFFER_KEY_ID` / `APPSTORE_OFFER_KEY`，券才会真正发放生效。
